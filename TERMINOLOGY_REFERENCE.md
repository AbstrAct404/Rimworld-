# Aya 人工种族汉化术语对照

本表用于游戏内文本、模组简介与 Steam 创意工坊简介的统一校对。RimWorld
原版已有译名时，必须采用原版简体中文用语；模组专名则以本项目的固定译名为准。

## RimWorld 原版术语

| 原文/内部名 | 固定译文 |
|---|---|
| Psychic sensitivity / PsychicSensitivity / 精神感応性 | 心灵敏感度 |
| General labor speed / GeneralLaborSpeed / 一般作業速度 | 普通工作速度 |
| Melee hit chance / MeleeHitChance | 近战命中率 |
| Shooting accuracy / ShootingAccuracyPawn | 射击精度 |
| Mental break / 精神崩壊 | 精神崩溃 |
| Immunity gain speed / 免疫獲得速度 | 免疫速度 |
| quadrum / season（RimWorld 时间单位） | 象限 |

不要把 `Psychic sensitivity` 简写成“敏感度”，也不要使用“灵能敏感度”“精神敏感度”
等近义写法。
