# [Aya] 人工种族简体中文整合汉化包 v1.6

Aya 人工种族系列简体中文整合汉化包。

可只安装自己使用的原模组，无需安装整合包支持的全部种族。请将本汉化置于所有 Aya 原模组之后加载；不要与对应的独立汉化包同时启用。

当前收录 21 个原模组，共合并 3020 条游戏文本。
