# [Aya] Xenoorca Race_zh 人工种族汉化 v1.6

本 MOD 新增泽诺奥卡种族、专属装备及相关派系。

【新增种族】
・泽诺奥卡

【泽诺奥卡的特点】
・寿命较长，擅长水域活动。
・较易受到疾病和毒素影响，免疫速度较慢。
・饥饿速度较慢。
・对高温及火焰攻击较为脆弱。
・受身体结构限制，只能穿戴部分装备。

【新增派系】
・奥西努斯旅
该派系永久敌对，部分成员会使用独特武器。通常在游戏开始约 5 年后才会发动袭击，实际强度可能因游戏设置和其他模组而变化。

【兼容联动】
・SwimmingKit
・Biomes! Islands
安装其中任一 MOD 后，泽诺奥卡可在海面以约 16 格/秒的速度移动。

【推荐但非必需的模组】
・Camera+

【注意】
・具体机制可能随原模组更新而调整。
