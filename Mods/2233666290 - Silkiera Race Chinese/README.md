# [Aya] Silkiera Race_zh 人工种族汉化 v1.6

本 MOD 新增希尔基耶拉种族、专属装备及相关派系。

【新增种族】
・希尔基耶拉

【如何招募】
・接纳流亡者，或营救逃生舱中的幸存者。
・从黑市商人或奴隶商人处购买（中等出现概率）。

【新增派系】
・亚人工业
该派系默认友好；若转为敌对，其袭击规模与投入的兵力通常高于一般派系。

【兼容联动】
・Nearmare Race
安装后，希尔基耶拉可穿戴部分尼亚梅尔专属装备。

【推荐但非必需的模组】
・Camera+

【希尔基耶拉的特点】
・寿命约为人类的 1.5 倍。
・基础身体能力略低于人类。
・较易受到疾病和毒素影响，免疫速度也较慢；随着年龄及“人工种族”等级提高，这些弱点会逐步缓解。
・饥饿速度略慢。
・不受技能等级影响的基础工作速度略快。

【注意】
・具体机制可能随原模组更新而调整。
・与原版体型服装不兼容，无法穿戴。
