# [Aya] Qualeela Race_zh 人工种族汉化 v1.6

本 MOD 新增夸莉拉种族、专属家畜、装备与相关派系。

夸莉拉属于首领级人工种族，能力强大，但招募和维持她们需要大量资源。建议在殖民地经济与防御稳定后再考虑接纳。

【新增种族】
・夸莉拉
・夸莉拉·凯特尔

【新增派系】
・恩特莱切亚
该派系初始敌对，但可以改善关系。通常在游戏开始约 5 年且殖民地财富达到约 450000 后才会发动袭击；其他 Aya 系列种族也可能混编进其袭击队伍。

【共同特点】
・拥有单次伤害上限，并免疫火焰伤害。
・能够使用“王权礼装”及多种专属能力。
・个体价值和维持成本都很高。

【夸莉拉的特点】
・生命与心灵敏感度约为人类的 1.5 倍。

【夸莉拉·凯特尔的特点】
・生命约为人类的 2 倍，心灵敏感度约为人类的 1.5 倍。
・除一般劳动外，多数工作速度较慢。
・只能使用特定武器，但拥有更强的专属能力。

【注意】
・该模组会显著提高后期战斗强度。
・具体机制可能随原模组更新而调整。
