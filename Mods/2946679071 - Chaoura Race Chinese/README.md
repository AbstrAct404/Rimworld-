# [Aya] Chaoura Race_zh 人工种族汉化 v1.6

本 MOD 新增混沌灵种族、专属装备与永久敌对派系。

【新增种族】
・混沌灵

【混沌灵的特点】
・寿命约为人类的 2 倍，市场价值为 5000。
・能力会随所在位置的亮度发生显著变化。
・心灵敏感度为 0，不会燃烧。
・身体耐久度很高，但没有额外的物理护甲。
・只能使用专属武器。
・极高概率拥有“夜猫子”特性。
・身处黑暗或看到尸体时不会产生心情惩罚。

【亮度对能力的影响】
・亮度 100%：身体能力降低，受到的伤害增加。
・亮度 0%：身体能力、恢复能力与减伤效果增强，并可免疫射击伤害。
・仅限敌方混沌灵：在亮度 0% 时受到射击，有中等概率传送至攻击者身边并反击。
能力取决于实际亮度，而非昼夜时间；使用照明可有效削弱其强化效果。

【新增派系】
・死伽屋
该派系永久敌对，通常在游戏开始约 5 年后才会来袭。其世界地图上没有可见据点，部队规模较小但个体精锐，主要使用近战。成员倒地时通常会撤离或消失；若被直接击杀则会留下尸体。

【新增事件】
・相伴深渊（游戏开始约 1 年后可能发生）
一名混沌灵会强制加入殖民地。

【应对建议】
照亮战场，避免在完全黑暗中与敌对混沌灵交战。

【注意】
・具体机制可能随原模组更新而调整。
