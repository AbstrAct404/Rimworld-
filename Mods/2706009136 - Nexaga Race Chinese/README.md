# [Aya] Nexaga Race_zh 人工种族汉化 v1.6

本 MOD 新增人工种族“涅克萨迦”。

【新增种族】
・涅克萨迦（Nexaga）

【重要说明】
安装后，涅克萨迦会混入原版派系。为确保世界生成与派系成员配置正常，建议安装后开始新游戏。

【如何招募】
・从袭击者中俘虏并招募。
・接纳流亡者，或营救逃生舱中的幸存者。
・从黑市商人或奴隶商人处购买（出现概率较高）。

【涅克萨迦的特点】
・寿命约为人类的 2 倍，基础能力与人类相近。
・没有召唤或增益等主动特殊能力，也没有装备限制。
・生成时会随机获得一种“混血种（人工种族名）”状态，并按血统获得相应强化。

【推荐但非必需的模组】
・Camera+
・Hats Display Selection / [KV] Show Hair With Hats or Hide All Hats
・HD Pawn Rendering / Pawn Render Patch

【注意】
・具体机制可能随原模组更新而调整。
