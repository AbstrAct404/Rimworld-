# [Aya] Littluna Race_zh 人工种族汉化 v1.6

本 MOD 新增莉特露娜种族、专属装备与相关事件。

【新增种族】
・莉特露娜

【莉特露娜的特点】
・寿命约为人类的 2 倍。
・移动速度略快于人类。
・身体耐久度较低，容易受到疾病和毒素影响，免疫速度也较慢。
・体格与小型动物相近。
・心灵敏感度较高，但精神崩溃阈值较低，因此相对不易发生精神崩溃。
・每小时会为附近单位施加一次增益效果。
・拥有强力召唤能力，但冷却时间长达 1 年；召唤物还会激怒附近敌人，使用前请确认周围环境安全。

【新增事件】
・幼年决战种（游戏开始 30 天后可能发生）
一名莉特露娜会来到殖民地附近，并在数小时后因体力不支而倒下；届时可将其俘获并尝试招募。

【注意】
・具体机制可能随原模组更新而调整。
