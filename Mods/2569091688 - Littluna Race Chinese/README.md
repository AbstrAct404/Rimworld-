# [Aya] Littluna Race_zh 人工种族汉化 v1.6

本 MOD 新增伪神族种族、专属装备与相关事件。

【新增种族】
・伪神族

【伪神族的特点】
・寿命约为人类的 2 倍。
・移动速度略快于人类。
・身体耐久度较低，容易受到疾病和毒素影响，免疫速度也较慢。
・体格与小型动物相近。
・心灵敏感度较高，但精神崩溃阈值较低，因此相对不易发生精神崩溃。
・每小时会为附近单位施加一次增益效果。
・拥有强力召唤能力，但冷却时间长达 1 年；召唤物还会激怒附近敌人，使用前请确认周围环境安全。

【新增事件】
・年幼的“决战种”（游戏开始 30 天后可能发生）
一名拥有强大却不稳定召唤能力的年幼伪神族会加入殖民地。她因长途跋涉而虚弱，可能很快因体力不支倒下；休息数天后即可恢复。

【注意】
・具体机制可能随原模组更新而调整。
