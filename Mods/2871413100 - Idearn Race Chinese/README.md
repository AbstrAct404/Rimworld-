# [Aya] Idearn Race_zh 人工种族汉化 v1.6

本 MOD 新增伊迪安种族、相关研究与后期制造系统。

这是面向资源和人力已经十分充足的殖民地设计的后期内容。研究耗时很长，不建议以尽快逃离星球为目标的流程使用。

【推荐但非必需的模组】
后期制造的个体使用较大的角色贴图，建议安装：
・Stylish Rim
・Camera+

【伊迪安人的共同特点】
・完成相关研究后，可通过“秘仪机”制造不同型号的伊迪安人。
・越高级的型号越强，并会解锁独特能力。
・每个象限必须使用一次特殊药物“本源药盒”，否则将停止活动并死亡。
・饥饿与疲劳增长较慢。
・心灵敏感度为 0，不会燃烧或流血，并能适应极端温度。

【特殊材料】
拆解机械族单位或摧毁飞船部件可获得制造所需的特殊材料；完成后续研究后，也可以自行生产部分材料。

【注意】
・本 MOD 面向后期高资源环境，早期启用可能造成较大负担。
・具体机制可能随原模组更新而调整。
