【新增种族】
・锢魂魔（Zoichor）

【锢魂魔的特点】
・初始状态受到力量封印，基础能力较弱。
・不会随时间自然成长，需要使用特殊物品“罗迪西安”分阶段强化。
・每个强化阶段都会解锁相应的特殊能力。
・拥有强大的身体再生能力。

【新增事件】
・古老的约定
游戏开始后有较高概率触发，且通常只发生一次。一名锢魂魔会加入殖民地；推进研究可帮助她逐步取回被封印的力量。
