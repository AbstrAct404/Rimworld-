# [Aya] Saclean Race_zh 人工种族汉化 v1.6

本 MOD 新增萨克琳种族及专属装备。

【新增种族】
・萨克琳

【如何招募】
・接纳流亡者，或营救逃生舱中的幸存者。
・从黑市商人或奴隶商人处购买（出现概率较高）。

【兼容联动】
・Nearmare Race
安装后，萨克琳可穿戴部分尼亚梅尔专属装备。

【萨克琳的特点】
・寿命约为人类的 2 倍，市场价值为 500。
・无法战斗，身体耐久度和移动速度也很低。
・饥饿速度较慢，疼痛耐受与免疫能力较高。
・作为囚犯或奴隶时不会主动越狱或反抗；界面仍可能显示相关概率。
・对恶劣环境造成的心情惩罚较不敏感。
・一般劳动速度较快。
・每个象限会产生一次丝质纤维，每 2 天产出一次乳汁。

【注意】
・具体机制可能随原模组更新而调整。
